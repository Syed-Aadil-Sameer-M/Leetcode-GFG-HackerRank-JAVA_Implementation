# [Java Exception Handling](https://www.geeksforgeeks.org/problems/java-exception-handling-1606978567/1?page=1&category=Java&difficulty=Medium&sortBy=latest)
## Medium
Given two integers a and b. Find the minimum value of a $ b where $ is any arithmetic operation like multiply( * ), Divide( / ), Addition( + ), Substraction( - ). Note: Use Exception Handling in this problem.
Examples:
Input: a = 5, b = -5
Output: -25
Explanation: 5+(-5) = 0, 5-(-5) = 10, 5*(-5) = -25, 5/(-5) = -1
Minimum of all is 5*(-5) = -25.
Input: a = 5, b = 0
Output: 0
Explanation: 5+0 = 5, 5-0 = 5, 5*0 = 0, 5/0 = "Exception Handling"
Minimum of all is 5*0 = 0.
Constraints:-1000 &nbsp;&lt;= &nbsp;a, b &nbsp;&lt;= &nbsp;1000
